## 📖 អំពី​គម្រោង​នេះ

QC Flash Cards គឺជាកម្មវិធីសិក្សាភាសាចិន-ខ្មែរ ដែលរចនាឡើងជាពិសេសសម្រាប់បុគ្គលិក QC (Quality Control) នៅក្នុងរោងចក្រ។ វាជួយឱ្យអ្នកសិក្សាពាក្យគន្លឹះ QC ដោយរបៀបងាយស្រួល និងមានប្រសិទ្ធភាព។

### ✨ មុខងារសំខាន់

- 📇 **Flash Cards អន្តរកម្ម** - ចុចដើម្បីត្រឡប់មើលការបកប្រែ
- 🔊 **ការបញ្ចេញសំឡេងស្វ័យប្រវត្តិ** - Google TTS ជាមួយ fallback
- ⚡ **ត្រួតពិនិត្យល្បឿន** - ជ្រើសរើសល្បឿនសំឡេង (1×, 0.6×, 0.4×)
- 🖼️ **រូបភាពគំរូ** - មើលរូបភាពបង្ហាញនៅខាងក្រោយ card
- 🔀 **ច្របល់ដោយចៃដន្យ** - ច្របល់ card ដើម្បីការរៀនល្អប្រសើរ
- ⌨️ **Keyboard Support** - ប្រើ ← → Space/Enter
- 📊 **របារវឌ្ឍនភាព** - តាមដានការរៀនរបស់អ្នក
- 📱 **Mobile Responsive** - ដំណើរការល្អនៅលើទូរស័ព្ទ

## 🎮 របៀបប្រើប្រាស់

### ការត្រួតពិនិត្យមូលដ្ឋាន

| សកម្មភាព | វិធី |
|---------|------|
| មើលការបកប្រែ | ចុចលើ card |
| Card បន្ទាប់ | ប៊ូតុង "បន្ទាប់ →" ឬ `→` key |
| Card មុន | ប៊ូតុង "← មុន" ឬ `←` key |
| បញ្ចេញសំឡេង | ប៊ូតុង 🔊 |
| ត្រឡប់ card | `Space` ឬ `Enter` key |

### មុខងារបន្ថែម

- **🔀 ច្របល់:** ច្របល់ Flash Cards ដោយចៃដន្យ
- **↻ ចាប់ផ្តើមឡើងវិញ:** ត្រឡប់ទៅ card ដំបូង
- **ត្រួតពិនិត្យល្បឿន:** ជ្រើសរើសល្បឿនសំឡេងតាមតម្រូវការ

## 📚 ពាក្យដែលរួមបញ្ចូល

កម្មវិធីនេះមានពាក្យ **49 ពាក្យ** ក្នុងប្រធានបទ៖

### 🔧 គុណភាព និង កំហុស (Quality & Defects)
- 起泡 (ពង/ចូលខ្យល់), 标纸不良 (តែមមិនល្អ), 修补痕 (ស្នាមជួសជុល)...

### 🏭 រោងចក្រ និង កន្លែងធ្វើការ (Factory & Workplace)
- 工厂 (រោងចក្រ), 车间 (រោងជាង), 办公室 (ការិយាល័យ)...

### 🚲 ផ្នែកកង់ (Bicycle Parts)
- 车架 (តួកង់), 前叉 (ឆ្ពោះមុខ), 链条 (ច្រវាក់)...

### 📊 QC & ការវាស់វែង (QC & Measurement)
- 测量值 (តម្លៃវាស់), 标准值 (តម្លៃស្តង់ដា), 合格 (ជាប់)...

### ⚡ កំហុសផ្សារ (Welding Defects)
- 开裂 (ប្រេះ), 焊破 (ផ្សាធ្លុះ), 漏焊 (ភ្លេចផ្សារ)...

## 🔧 ការតម្រូវតាមតម្រូវការ

### បន្ថែមពាក្យថ្មី

1. បើកឯកសារ `index.html`
2. រកមើលផ្នែក `database` array
3. បន្ថែមពាក្យថ្មីតាមទម្រង់៖
```javascript
const database = [
    // ... ពាក្យដែលមានស្រាប់
    { 
        cn: "新词",           // ពាក្យចិន
        km: "ពាក្យថ្មី",      // ការបកប្រែខ្មែរ
        img: "image.jpg"     // រូបភាព (optional)
    },
    // ...
];
```

### ប្តូររូបរាង និង ពណ៌

ប្តូរតម្លៃក្នុង `:root` CSS variables:
```css
:root {
    --bg:       #070b1a;    /* ពណ៌ផ្ទៃខាងក្រោយ */
    --primary:  #4f8ef7;    /* ពណ៌សំខាន់ */
    --accent:   #f4c430;    /* ពណ៌បន្ថែម */
    /* ... */
}
```
